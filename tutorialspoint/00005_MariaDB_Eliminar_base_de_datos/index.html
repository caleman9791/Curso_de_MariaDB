La creación o eliminación de bases de datos en MariaDB requiere privilegios, por lo general, solo dados a usuarios o administradores raíz. En estas cuentas, tiene dos opciones para eliminar una base de datos: el binario mysqladmin y un script PHP.

Tenga en cuenta que las bases de datos eliminadas son irrecuperables, por lo que ejerce cuidado al realizar esta operación. Además, los scripts PHP utilizados para la eliminación no le solicitan una confirmación antes de la eliminación.

PHP 5 y posterior puede funcionar con una base de datos MySQL usando:

  • Extensión mysqli (la "i" significa mejorar)
  • PDO (objetos de datos PHP)

Las versiones anteriores de PHP utilizaron la extensión MySQL. Sin embargo, esta extensión se vio en desuso en 2012.

¿Debo usar mysqli o pdo?

Si necesita una respuesta breve, sería "lo que quieras".

Tanto Mysqli como PDO tienen sus ventajas:

SEgun la documentalista de PHP, PDO funcionará en 12 sistemas de bases de datos diferentes, mientras que MySQLI solo funcionará con bases de datos MySQL.
https://www.php.net/manual/en/pdo.drivers.php

Entonces, si tiene que cambiar su proyecto para usar otra base de datos, PDO facilita el proceso. Solo tiene que cambiar la cadena de conexión y algunas consultas. Con MySQLI, deberá reescribir todo el código, consultas incluidas.

Ambos están orientados a objetos, pero MySQLI también ofrece una API de procedimiento.

Ambos apoyan declaraciones preparadas. Las declaraciones preparadas protegen de la inyección SQL y son muy importantes para la seguridad de las aplicaciones web.
